# Rest Countries API App

![Preview](preview.jpg)

This is a **Rest Countries API App** built to practice working with the **[REST Countries API](https://restcountries.com/)**, using **fetch**, **DOM manipulation**, and other JavaScript concepts like **URLSearchParams for query handling, deep object access, event handling, and array methods**. The project includes a **responsive layout** using **CSS Grid** and **Flexbox**, while following good **HTML conventions** and **accessibility rules**.

This project was originally a **[Frontend Mentor](https://www.frontendmentor.io/)** challenge.

## Features

- Search and filter countries by name and region
- Responsive design for desktop and mobile
- Dark mode toggle that persists across sessions
- Clickable country cards leading to detailed country page

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Variables)
- JavaScript (fetch, DOM manipulation, URLSearchParams, deep object access, event handling, array methods)
